افزونه جستجوی محصولات المنتور - یک ویجت زیبا برای جستجوی محصولات به صورت AJAX در المنتور

== Description ==

این افزونه یک ویجت آیکون جستجو به المنتور اضافه می‌کند که:

* با کلیک روی آیکون، صفحه کمی تیره می‌شود
* یک باکس جستجوی زیبا از پایین صفحه به بالا حرکت می‌کند
* یک فیلد ورودی بزرگ و زیبا برای جستجوی محصولات نمایش داده می‌شود
* با تایپ هر کاراکتر، نتایج به صورت آنی نمایش داده می‌شوند
* محصولات مرتبط با دقت بالا نمایش داده می‌شوند
* قابلیت اسکرول اتوماتیک برای نتایج زیاد

== نیازمندی‌ها ==

* وردپرس 5.0 یا بالاتر
* المنتور 2.9.0 یا بالاتر
* ووکامرس 3.0 یا بالاتر

== نصب ==

1. فایل ZIP افزونه را دانلود کنید
2. به بخش افزونه‌های وردپرس بروید و روی "بارگذاری افزونه" کلیک کنید
3. فایل ZIP دانلود شده را انتخاب کنید
4. افزونه را فعال کنید

== استفاده ==

1. یک صفحه را با المنتور ویرایش کنید
2. ویجت "آیکون جستجوی محصولات" را در پنل المنتور پیدا کنید
3. آن را به مکان مورد نظر خود بکشید
4. تنظیمات ویجت را در پنل المنتور سفارشی کنید

== تنظیمات ==

=== تنظیمات آیکون ===
* اندازه آیکون: تعیین اندازه آیکون جستجو
* رنگ آیکون: انتخاب رنگ آیکون جستجو
* رنگ هاور آیکون: انتخاب رنگ آیکون هنگام هاور

=== تنظیمات باکس جستجو ===
* متن نگهدارنده جستجو: متن پیش‌فرض فیلد جستجو
* زیرنویس جستجو: متن زیر فیلد جستجو
* رنگ اورلی: انتخاب رنگ و شفافیت پس‌زمینه تیره
* پس‌زمینه باکس جستجو: انتخاب رنگ پس‌زمینه باکس جستجو

=== استایل ورودی ===
* پس‌زمینه ورودی: انتخاب رنگ پس‌زمینه فیلد ورودی
* رنگ متن ورودی: انتخاب رنگ متن فیلد ورودی
* تایپوگرافی: تنظیم اندازه فونت، وزن و غیره برای فیلد ورودی
* حاشیه: سفارشی‌سازی حاشیه فیلد ورودی
* شعاع حاشیه: تنظیم شعاع حاشیه فیلد ورودی
* پدینگ: تنظیم فاصله داخلی فیلد ورودی

== عیب‌یابی ==

اگر ویجت در لیست ویجت‌های المنتور نمایش داده نمی‌شود:

1. مطمئن شوید که المنتور و ووکامرس نصب و فعال هستند
2. از افزونه غیرفعال کنید و سپس دوباره فعال کنید
3. کش المنتور را پاک کنید (المنتور > ابزارها > کش بازسازی)
4. کش مرورگر خود را پاک کنید
5. فهرست ویجت‌ها را در المنتور جستجو کنید (جستجوی "محصولات" یا "جستجو")
